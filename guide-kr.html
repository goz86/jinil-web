주문 상태 흐름

주문은 아래 4단계 상태로 관리됩니다. 상태별로 색상이 다르게 표시됩니다.

📝 신규
→

→
📦 부분출고
→
✅ 출고완료
상태 의미 다음 단계
📝 신규 주문이 막 접수된 상태. 아직 처리 전. 주문 확인 후 "대기"로 변경하거나 바로 출고 처리
⏳ 대기 확인 완료, 출고 준비 중인 상태 모바일 앱으로 송장 촬영 후 출고 처리
📦 부분출고 일부 품목만 출고된 상태 나머지 품목 출고 후 "출고완료"로 전환
✅ 출고완료 모든 품목이 출고된 최종 상태 완료 — KakaoTalk / SMS로 거래처에 자동 알림 발송
💡 부분출고란? 주문한 품목 중 일부만 먼저 보내는 경우입니다. 나머지는 다음 번에 추가 출고할 수 있으며, 모두 출고되면 자동으로 "출고완료"로 변경됩니다.
⚡ 주요 기능

메뉴별 기능 안내

웹 화면 상단 탭별로 제공되는 기능을 확인하세요.

📋

주문 관리

거래처별 주문을 생성·수정하고 상태를 추적합니다. 검색·날짜 필터로 원하는 주문을 빠르게 찾을 수 있습니다.

기본으로 최근 90일 주문이 표시되며, "이전 데이터 불러오기" 버튼으로 더 오래된 내역도 조회할 수 있습니다.

👥

고객 관리

거래처 목록을 관리합니다. 각 거래처에는 담당자 이름, 전화번호, 사업자번호, 주소 등의 정보를 입력할 수 있으며 이 정보가 거래명세서에 자동으로 반영됩니다.

💰

지출 관리

택배비·구매비 등 운영 비용을 날짜·항목별로 기록합니다. 수입과 지출을 비교하여 손익 현황을 확인할 수 있습니다.

📑

거래명세서 자동 생성

출고 처리 시 PDF 거래명세서가 자동으로 생성됩니다. 공급자·거래처 정보, 품목·수량·단가·합계가 포함된 A4 형식의 문서입니다.

Google Docs 링크로 거래처에 전달됩니다.

🔔

실시간 알림

모바일 앱에서 출고 처리 시 웹의 알림 벨(🔔)에 즉시 숫자가 표시됩니다. 클릭하면 최근 48시간의 출고 내역을 확인할 수 있으며, "이전 데이터 불러오기"로 더 오래된 알림도 조회 가능합니다.

💳

결제 요청

직원이 경비 승인을 요청하면 관리자에게 실시간 알림이 전송됩니다. 관리자는 승인 또는 거절을 선택할 수 있으며, 결과는 즉시 직원에게 전달됩니다.

📱 모바일 앱

모바일 출고 처리 순서

Android 앱에서 출고를 처리하는 단계별 방법입니다.

1

앱 실행 후 로그인

관리자에게 받은 이메일·비밀번호로 로그인합니다. 한 번 로그인하면 자동으로 유지됩니다.

2

거래처 선택

출고할 거래처를 목록에서 선택합니다. 이름으로 검색도 가능합니다.

3

송장 촬영

박스에 붙은 송장 라벨을 카메라로 촬영합니다. 여러 장 촬영 가능합니다.

💡 사진 2장 이상 촬영 후 뒤로 가면 자동으로 공유 팝업이 열립니다.
4

출고할 품목 체크

주문 목록에서 이번에 보낼 품목에 체크 표시를 합니다. 일부만 선택하면 자동으로 "부분출고"로 처리됩니다.

5

송장번호 입력

택배 송장번호를 입력합니다. 롯데택배는 13자리, 경기택배는 16자리입니다. 앱이 자동으로 택배사를 구분합니다.

6

KakaoTalk 또는 SMS 발송

발송 방법을 선택합니다. KakaoTalk으로 보내면 사진 + 거래명세서 링크 + 품목 정보가 함께 전송됩니다.

📱 KakaoTalk 권장 — 사진을 직접 첨부하여 거래처가 바로 확인할 수 있습니다.
7

출고 완료 처리

발송 완료 후 "출고 완료 처리" 버튼을 누릅니다. 웹의 관리자·직원에게 실시간 알림이 전송되고 주문 상태가 자동으로 업데이트됩니다.

🚚 택배사 자동 구분: 송장번호 16자리 → 경기택배 / 13자리 → 롯데택배로 자동 인식됩니다.
🔔 알림 시스템

알림 종류 및 동작 방식

시스템은 두 가지 종류의 실시간 알림을 제공합니다.

알림 종류 발생 시점 수신 대상 확인 방법
📦 출고 알림 모바일 앱에서 출고 처리 시 관리자 + 직원 화면 우상단 🔔 벨 클릭
📝 주문 생성 알림 직원이 웹에서 주문 생성 시 관리자만 화면 우상단 🔔 벨 클릭
💳 결제 요청 알림 직원이 결제 요청 생성 시 관리자만 화면 우상단 💳 버튼 클릭
⚠️ 알림 초기화: 🔔 벨을 클릭하면 읽음 처리되어 숫자가 0으로 초기화됩니다. 이는 숫자(뱃지)만 없어지는 것이며, 실제 주문·출고 데이터는 삭제되지 않습니다.

FAQ

사용 중 자주 발생하는 질문들에 대한 답변입니다.

질문 답변
📱 앱에서 출고했는데 웹에 바로 반영이 안 됩니다 네트워크 상태에 따라 최대 1~2초 지연될 수 있습니다. 페이지를 새로고침하거나 잠시 기다려 주세요.
🔔 알림 숫자가 계속 쌓입니다 🔔 벨 버튼을 클릭하면 팝업이 열리면서 자동으로 읽음 처리됩니다. "알림 모두 지우기" 버튼으로도 초기화 가능합니다.
📋 오래된 주문이 목록에 없습니다 기본적으로 최근 90일 데이터만 표시됩니다. 주문 탭 하단의 "이전 데이터 불러오기" 버튼을 클릭하면 더 오래된 내역을 불러올 수 있습니다.
📑 거래명세서 PDF 링크가 너무 깁니다 링크가 길더라도 거래처에서 클릭하면 정상적으로 열립니다. KakaoTalk으로 보낼 경우 사진이 직접 첨부되어 링크 없이도 확인 가능합니다.
📷 사진을 여러 장 찍고 싶습니다 앱에서 사진을 촬영 후 뒤로 가면 다시 촬영할 수 있습니다. 2장 이상이 되면 자동으로 공유 팝업이 열립니다.
🚚 경기택배 / 롯데택배를 어떻게 구분하나요? 송장번호 자릿수로 자동 구분됩니다. 16자리 → 경기택배, 13자리 → 롯데택배.
🔐 비밀번호를 잊어버렸습니다 관리자에게 문의하여 비밀번호 재설정을 요청하세요.
